# Photo Gallery

Drop your photos here! 📸

## Instructions:
1. Add your photos to this folder
2. Supported formats: .jpg, .jpeg, .png, .gif, .webp
3. Photos will be automatically loaded and displayed in the gallery
4. Each photo will be displayed in a romantic polaroid frame

## Tips:
- Use high-quality photos for best results
- Landscape photos work best in the polaroid frames
- The gallery will automatically handle different image sizes
