# Music Files

Add your romantic instrumental music files here:

## Required Files:
- `romantic-instrumental.mp3` - Main romantic background music
- `romantic-instrumental.ogg` - Fallback format for better browser compatibility

## Recommended Music:
- Soft piano melodies
- Romantic guitar instrumentals
- Classical love songs
- Ambient romantic music

## File Formats:
- MP3 (primary format)
- OGG (fallback format)

## Usage:
The music will autoplay when the website loads (if browser allows) and can be controlled by the music player button in the bottom-right corner.

Place your music files in this directory to enable the background music feature.
